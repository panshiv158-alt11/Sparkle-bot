🔧 TROUBLESHOOTING:

Problem: Videos not showing
  ➜ Check if videos are in the folder
  ➜ Use start.bat or node server.js
  ➜ Verify exact filename spelling
  ➜ Run copy_videos.bat

Problem: QR code not generating
  ➜ Refresh page (F5)
  ➜ Wait 2-3 seconds
  ➜ Check browser console (F12)

Problem: Port already in use
  ➜ Edit server.js, change PORT to 8080
  ➜ Or use Python server instead

Problem: Website looks broken
  ➜ Clear browser cache (Ctrl+Shift+Del)
  ➜ Refresh page (Ctrl+F5)
  ➜ Try different browser
