⚠️ TROUBLESHOOTING:

VIDEOS NOT PLAYING?
  ➜ Run: copy_videos.bat
  ➜ Check file names (EXACT match needed)
  ➜ Use: start.bat (not direct file open)

WEBSITE LOOKS WEIRD?
  ➜ Refresh page: Ctrl+F5
  ➜ Clear cache: Ctrl+Shift+Del
  ➜ Try different browser

QR CODE NOT SHOWING?
  ➜ Refresh page: F5
  ➜ Wait 2-3 seconds
  ➜ Check browser console: F12

PORT ALREADY IN USE?
  ➜ Edit: server.js
  ➜ Change: PORT = 3000 to PORT = 8080
