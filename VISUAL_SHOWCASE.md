## 🎉 VISUAL HIERARCHY

**Size hierarchy:**

```
H1 Hero Headline: 3.5rem (huge!)
    ↓
H2 Section Titles: 2.5rem (large)
    ↓
H3 Card Titles: 1.3-1.5rem (medium)
    ↓
Body Text: 1rem (readable)
    ↓
Small Text: 0.75-0.9rem (meta info)
```

**Color hierarchy:**

```
Dark Blue: Headlines, primary text
Cyan: Secondary CTAs, accents
Orange: Primary CTAs, urgency
Light: Background, breathing room
```

**Importance by position:**

```
TOP: Hero section (immediate attention)
ABOVE FOLD: Free demo classes
MIDDLE: Learning paths
BELOW: Courses
LOWER: Videos, testimonials
FOOTER: Contact info
FIXED: Floating CTA (always important)
```
